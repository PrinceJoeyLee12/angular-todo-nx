## Requirements

### node version 18 using nvm

```
nvm install v18
```

```
nvm use v18
```

### nx install

```
npm install -g nx@latest
```

## Applications

### Todo app (not standalone) - Run the following

```
npm install
```

```
nx serve angular-todo-two
```

### Todo app (standalone) - Run the following

```
npm install
```

```
nx serve angular-todo
```
